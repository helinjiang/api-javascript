`Math.pow(base, exponent)` 函数返回基数（`base`）的指数（`exponent`）次幂。

## 定义和用法

### 语法

`Math.pow(base, exponent) `

| 参数 | 描述 |
| --- | --- |
| _base_ | 基数 |
| _exponent_ | 指数 |

### 返回值

`base` 的 `exponent` 次幂。

### 说明

由于 `Math.pow()` 是 `Math` 中的一个静态方法，你应该通过 `Math.pow()` 调用。（`Math` 不是构造器）

## 实例

```javascript

    Math.pow(2,3); // 8

```
