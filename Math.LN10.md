`Math.LN10` 属性表示 `10` 的自然对数，约等于 `2.302`。

`Math.LN10` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
