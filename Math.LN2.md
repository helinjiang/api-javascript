`Math.LN2` 属性表示 `2` 的自然对数，约等于 `0.693`。

`Math.LN2` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
