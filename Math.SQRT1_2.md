`Math.SQRT1_2` 属性表示表示 `1/2` 的平方根，约等于 `0.707`。

`Math.SQRT1_2` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
