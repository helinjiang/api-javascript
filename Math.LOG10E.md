`Math.LOG10E` 属性表示以 `10` 为底数，`e` 的对数，约等于 `0.434`。

`Math.LOG10E` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
