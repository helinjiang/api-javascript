`String.prototype.trim()` 方法删除字符串两端的空白字符并返回。

## 定义和用法

### 语法

`stringObject.trim()`

### 返回值

一个新的字符串。

## 实例

``` javascript

    var str = "   foo  ";
    console.log(str.trim()); // "foo"
    console.log(str); // "   foo  "，不会改变原字符串    

```
