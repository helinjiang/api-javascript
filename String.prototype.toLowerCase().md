`String.prototype.toLowerCase()` 方法用于把字符串转换为小写。

## 定义和用法

### 语法

`stringObject.toLowerCase()`

### 返回值

一个新的字符串，在其中 `stringObject` 的所有大写字符全部被转换为了小写字符。

## 实例

``` javascript

    var str = "Hello World!";
    console.log(str.toLowerCase()); // "hello world! "
    console.log(str); // "Hello World!"，不会改变原字符串    

```
