`Number.prototype` 属性表示 `Number` 构造函数的原型。

`Number.prototype` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |

## 定义和用法

### 描述

在JavaScript中，所有的 `Number` 实例继承自 `Number.prototype`；修改 `Number` 构造函数的原型对象会影响到所有 `Number` 实例。
