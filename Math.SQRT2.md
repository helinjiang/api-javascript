`Math.SQRT2` 属性表示 `2` 的平方根，约等于 `1.414`。

`Math.SQRT2` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
