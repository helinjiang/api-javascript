`Math.ceil(x)` 返回一个大于或等于数值 "`x`" 的最小整数。

## 定义和用法

### 语法

`Math.ceil(x)`

| 参数 | 描述 |
| --- | --- |
| _x_ | 必需。必须是一个数值。 |

### 返回值

大于等于 `x`，并且与它最接近的整数。

### 说明

由于 `Math.ceil()` 是 `Math` 中的一个静态方法，你应该通过 `Math.ceil()` 调用。（`Math` 不是构造器）

## 实例

```javascrpt
    Math.ceil(.95);   // 1
    Math.ceil(4);     // 4
    Math.ceil(7.004)  // 8
    Math.ceil(-7.004) // -7
```
