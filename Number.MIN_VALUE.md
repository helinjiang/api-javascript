`Number.MIN_VALUE` 属性表示在 JavaScript 里所能表示的最小正值。

`Number.MIN_VALUE` 属性的属性特性：

<dl class="dl-horizontal">

<dt>writable</dt>

<dd>`false`</dd>

<dt>enumerable</dt>

<dd>`false`</dd>

<dt>configurable</dt>

<dd>`false`</dd>

</dl>

## 定义和用法

### 描述

`MIN_VALUE` 属性值最接近于 `0` 的正值，而不是最小的负值。小于 `MIN_VALUE` 的值（"underflow values"）将会转换为 `0`。
