`Math.PI` 属性表示一个圆的周长与直径的比例，约等于 `3.14159`。

`Math.PI` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
