`Math.E` 属性表示自然对数 `e` 的底数（或称为基数），约等于 `2.718`。

`Math.E` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
