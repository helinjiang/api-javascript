`Number.MAX_VALUE` 属性表示在 JavaScript 里所能表示的最大数值。

`Number.MAX_VALUE` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |

## 定义和用法

### 描述

`MAX_VALUE` 是 JavaScript 里所能表示的最大数值。大于 `MAX_VALUE` 的值代表 "Infinity"。
