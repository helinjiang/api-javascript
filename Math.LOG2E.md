`Math.LOG2E` 属性表示以 `2` 为底数，`e` 的对数，约等于 `1.442`。

`Math.LOG2E` 属性的属性特性：

|  属性         | 值     |
| ------------  | -------|
|  writable     | false  |
|  enumerable   | false  |
|  configurable | false  |
